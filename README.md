# PummiRalli

[![Build Status](https://travis-ci.org/AradoLabs/pummiralli-server.svg?branch=master)](https://travis-ci.org/AradoLabs/pummiralli-server)

## Requirements

- nvm (or just use the node version specified in `.nvmrc`)

### Run server

```bash
yarn start
```

### Run test client
```bash
yarn test-client
```
